Let's Learn Spanish

A fun little web app developed for anyone seeking to learn a few words in Spanish. This is an experiment in utilizing the "spaced repetition method" of learning. The user is given a list of words to individually translate. With each guess, the user is provided feedback of total times they are correct/incorrect for each word.
